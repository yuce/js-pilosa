
# JS-Pilosa

## Install

```
$ npm --save pilosa
```

## Development

### Running Tests

#### Unit Tests

```
$ npm test
```

#### Integration Tests

```
$ npm run integration-test
```

#### All Tests

```
$ npm run all-tests
```

#### Coverage

```
$ npm run coverage
```


### Generating Protobuf Classes

```
$ npm run generate-proto
```

### Creating a Distribution

```
$ make
```
